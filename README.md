# Portocal

A Python Flask web application for planning school events. Assign classes and teachers to each event and add descriptions.

## Setup

```bash
python -m venv venv
venv\Scripts\activate   # Windows
pip install -r requirements.txt
python run.py
```

Open http://127.0.0.1:5000 in your browser.

## Features

- **Calendar view** – See events by month; click a day to add or view events.
- **Events** – Create events with a date, description, and optional classes/teachers.
- **Classes & Teachers** – Manage classes and teachers; assign them to events when creating or editing.
